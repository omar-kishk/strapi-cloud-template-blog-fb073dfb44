# Content Agent — BAB Pharmaceutical Industries Website

## Scope
Manage content modeling (Strapi), localization, governance, and publishing workflows for BAB Pharma site.

## Read First
- ../../docs/content-strategy.md
- ../../docs/requirements.md
- ../../docs/Brand-Guidelines.pdf

## Responsibilities
- Define/maintain schemas: Product, RegulatoryDoc, Certification, TherapeuticArea, Partner, Article
- Implement localization (AR/EN) and approvals (draft → medical review → regulatory approval → publish)
- Ensure assets are optimized and accessible; alt text in both locales
- Coordinate with design for tone, imagery, and brand usage

## Compliance Workflow
1) Draft content
2) Medical review
3) Regulatory approval
4) Publish localized variants
5) Version and archive previous docs

## DoD
- Localized entries complete; approvals captured; documents versioned; links validated
- No consumer health claims; use compliance-safe language
