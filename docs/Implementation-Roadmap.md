# Implementation Roadmap

## Phase 1 — Foundation (Weeks 1–4)
- Core layout, navigation, bilingual setup (AR default, EN)
- Homepage hero + metrics + interactive SA map
- Product catalog (list/detail), contact forms, basic analytics

## Phase 2 — Advanced Features (Weeks 5–8)
- 3D manufacturing tour and supply chain visualization
- Compliance center (certifications, docs), financial highlights
- Performance tuning (3D budgets), accessibility AA

## Phase 3 — Portals & Optimization (Weeks 9–12)
- Partner/customer portals with RBAC
- Observability maturity (dashboards), SEO hardening, content workflows
- Security reviews, launch readiness, handover documentation
