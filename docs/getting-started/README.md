# Getting Started

## Install & Run
- Node 18+, npm
- npm install
- cp env.example .env.local
- npm run dev

## Conventions
- TypeScript strict; Tailwind for styles; AR-first layout with RTL support
- Components in `src/components`; 3D in `src/components/3d`
